## Building the project

```
make build
```

## Running the first node

```
make run
```

## Running Followers

```
make runfollower
```

NOTE: this will run the Follower with listenaddr 4000, if you want it to run with another port see `Makefile`

## Testing

you can test the `GET` and `SET` Commands using

```
make runset
make runget
```
